# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-11-05

### Added

- Initial release of Dintero Rust SDK
- Complete implementation of Dintero Checkout API
  - Session management
  - Card token operations
  - Transaction handling
- Complete implementation of Dintero Orders API
  - Order creation and management
  - Order status tracking
  - Order operations
- Complete implementation of Dintero Payments API
  - Payment captures
  - Payment refunds
  - Payment voiding
  - Payment operations
- Complete implementation of Dintero Accounts API
  - Account profiles
  - Account management
  - Profile operations
- Complete implementation of Dintero Loyalty API
  - Loyalty programs
  - Rewards management
  - Points tracking
- Complete implementation of Dintero Insights API
  - Analytics and reporting
  - Data queries
- Feature-based modular architecture
- Comprehensive examples for all modules
- Type-safe request and response models
- Builder patterns for ergonomic API usage
- Async/await support
- Flexible authentication methods
- Environment variable configuration
- Full Rust documentation
